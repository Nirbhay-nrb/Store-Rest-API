# Store Rest API
## Description
This is a code along Rest API deployed on Heroku
## Technology used:
Flask using python<br>
### Dependencies: 
Flask-RESTful
<br>Flask-JWT
<br>Flask-SQLAlchemy

### URL:
``` https://stores-rest-api-flask-1.herokuapp.com ```
### End points:
```
/items
/item/<name>
/stores
/store/<name>
/register
/login
```
